# Time Series Analysis of Technology Stocks

Conducted time series analysis of stock data for Amazon, Google, Apple, and Microsoft using Pandas, NumPy, Matplotlib, and Seaborn. Analyzed price trends, trading volumes, and daily returns, performed multivariate correlation analysis, and calculated Value at Risk (VaR) to evaluate portfolio risk and uncover relationships between leading technology stocks.
